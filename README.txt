


HOW TO OPEN THIS PROGRAM

1. In PowerShell / a Linux distro, cd into repository folder

use command 

python -m venv venv

followed by

.\venv\Scripts\activate

and then

python init_database.py

followed by

python app.py

The app should load on localhost:8000

From there you can perform CRUD operations on people and notes, while also having use of a data accessor to 
see your people and notes as JSON schema.
